3.Ans: There are three different types of HTML lists:

1. Ordered List or Numbered List (ol)
2. Unordered List or Bulleted List (ul)
3. Description List or Definition List (dl)

Ordered lists:

  1. This is a list item.
  2. This is a second list item.
  3. Three list items right here.

Unorder List

  • This is a list item.
  • This is a second list item.
  • Three list items right here.

Description Lists

Coffee
- black hot drink
Milk
- white cold drink
Mango
- Red mango
